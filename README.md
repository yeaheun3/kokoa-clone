# GitHub 올리기

```
git add . # 현재까지 작성한 내용 스테이징(로컬에서 변경사항 올리는 방법)
git commit -m "쓰고싶은 commit 내용" # 스테이징 한 것을 저장하는 방법
git push origin main # commit 내용을 원격 저장소에 올리는 방법

```
